# 헬스 블로그 완전 자동화 설계

**날짜:** 2026-04-02  
**프로젝트:** health-blog (health-blog-flax.vercel.app)  
**목표:** AI 포스트 자동 생성 + SEO 최적화 + 수익화 + 텔레그램 알림

---

## 전체 구조

```
[GitHub Actions 스케줄러 - 매일 오전 9시]
         ↓
[pytrends로 오늘의 인기 건강 키워드 수집]
         ↓
[GPT-4o-mini로 SEO 최적화 포스트 생성]
  - 2000자 이상 롱폼 콘텐츠
  - E-E-A-T 원칙 반영
  - FAQ 섹션 자동 포함
  - 구조화 데이터(Schema.org) 자동 삽입
  - 쿠팡 파트너스 링크 자동 삽입
  - 메타 description 자동 생성
         ↓
[MDX 파일 생성 → GitHub 자동 커밋]
         ↓
[Vercel 감지 → 자동 빌드 & 배포]
         ↓
[sitemap.xml 갱신 → 구글 Search Console 알림]
         ↓
[텔레그램 봇으로 발행 완료 알림 전송]
```

---

## 1. 키워드 분석 (동적)

- **도구:** pytrends (구글 트렌드 비공식 라이브러리)
- **비용:** 무료
- **방식:** 건강/다이어트/보충제 카테고리에서 오늘 인기 키워드 수집
- **선택 기준:** 검색량 높고 경쟁도 낮은 키워드 1개 자동 선택

---

## 2. SEO 최적화 포스트 생성 (정적 프롬프트)

- **모델:** GPT-4o-mini (비용 최소화)
- **프롬프트 내재화 규칙:**
  - E-E-A-T (경험·전문성·권위·신뢰) 원칙
  - 검색 의도(정보형/구매형/비교형) 맞춤 구성
  - 핵심 키워드 제목, 소제목, 본문에 자연스럽게 배치
  - FAQ 섹션 포함 (구글 People Also Ask 대응)
  - 구조화 데이터(Schema.org Article) 자동 삽입
  - 2000자 이상 롱폼 콘텐츠
  - 내부 링크 자동 연결
- **생성 결과물:**
  - MDX 포스트 파일
  - 메타 title / description
  - OG 태그
  - 쿠팡 파트너스 링크 3~5개

---

## 3. 수익화

### 구글 애드센스
- `layout.tsx`에 애드센스 스크립트 삽입
- 포스트 상단, 중간, 하단 광고 위치 자동 배치
- 승인 조건: 포스트 10~20개 이상, 독창적 콘텐츠

### 쿠팡 파트너스
- 키워드별 쿠팡 상품 링크 매핑 테이블 관리
- AI 포스트 생성 시 관련 키워드 감지 → 자동 링크 삽입
- 포스트 하단 "관련 상품 보기" 섹션 자동 추가
- 수익: 클릭 후 구매 시 3~10% 수수료

---

## 4. SEO 기술 설정

- `sitemap.xml` 자동 생성 (Next.js App Router 내장)
- `robots.txt` 추가
- Google Search Console 연동
- 구조화 데이터(JSON-LD) 포스트마다 자동 삽입
- OG 이미지: Vercel OG (텍스트 기반, 무료)

---

## 5. 텔레그램 알림

- 텔레그램 봇 생성 (BotFather)
- GitHub Actions 마지막 단계에서 발행 완료 시 메시지 전송
- 알림 내용: 포스트 제목, 키워드, URL, 발행 시각
- Bot Token → GitHub Secrets 저장
- 비용: 완전 무료
